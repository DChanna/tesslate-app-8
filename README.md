- Backend Docs: http://localhost:8000/docs

### Option 2: Manual Setup

#### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

#### Frontend  
```bash
cd frontend
npm install
npm run dev
```
